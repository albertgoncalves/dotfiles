# dotfiles

### Install
 * [Conda](https://anaconda.org/)
 * [Nix](https://nixos.org/nix/)

---
### Prepare
 * `path/to/dotfiles/.env` exporting the following:

```
export GIT_URL="https://raw.githubusercontent.com/..."
export WD
WD=$(pwd)
```

---
### Invoke
(This may be a **little** destructive.)
```
./main
```
